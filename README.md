Zenith is a production-ready, hybrid Machine Learning underwriting system designed specifically for the Indian banking sector. It bridges the gap between traditional rule-based banking guidelines and advanced predictive modeling, featuring a dynamic Flask frontend and custom financial heuristics.

## 🧠 Core Philosophy: The "Hybrid" Architecture

Standard "black-box" models often suffer from data blindness—blindly rejecting applicants based on single metrics like missing collateral. Zenith solves this by combining human financial intuition with Machine Learning precision. It utilizes a **4-Stage Hybrid Pipeline** tailored to Indian banking norms (like CIBIL scores):

1. **Institutional Knockout Rules (Stage 1):** Deterministic guardrails that instantly auto-reject impossible applications (e.g., severe debt-to-income imbalances or sub-50k INR incomes) before they confuse the AI.
2. **Custom Feature Engineering (Stage 2):** Translates raw financial data into "Smart Variables" based on banking logic. This includes categorizing *CIBIL Tiers* and calculating our custom *Income Rescue Power* (measuring if exceptional cash flow can cover missing hard assets).
3. **Machine Learning Prediction (Stage 3):** A multidimensional **Random Forest Classifier** evaluates the engineered features to calculate precise default probabilities based on historical Indian loan data.
4. **Intelligent Messaging (Stage 4):** A dynamic logic router that returns exact, plain-English explanations for *why* an application was approved or rejected (e.g., "Missing collateral, but strong credit history justified the risk").

## ⚙️ Installation & Setup

### Prerequisites
- **Python 3.8+**
- **pip** (Python package manager)

### 2. Install Dependencies
```bash
pip install pandas scikit-learn flask matplotlib seaborn joblib
```

### 3. Execute the Data Pipeline & Train Models
Ensure `loan_approval_dataset.csv` is located in the `data/` directory. Run the training script to bake your custom financial logic into the Random Forest model and generate the `.pkl` files:
```bash
python train_hybrid_model.py
```

*(Optional)* Run the exploratory data scripts to view the underlying CIBIL and Income distributions:
```bash
python data_exploration.py
python feature_importance.py
```

### 4. Launch the Enterprise Application
Start the Flask API backend:
```bash
python app.py
```
Once the server is running, open your web browser and navigate to:
**`http://127.0.0.1:5000`**
